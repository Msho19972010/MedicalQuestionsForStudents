# MedicalQuestionsForStudents
This is a Java-based medical knowledge quiz application built using Swing for the user interface. The app tests users on various medical topics, supports bilingual functionality (Slovak and English), and provides a modern and user-friendly design.It includes features such as:

    Bilingual Support: Switch between Slovak and English languages seamlessly.

    Interactive Quiz: Answer multiple-choice questions and get instant feedback.

    Score Tracking: Keep track of your score as you progress through the quiz.

    Mistake Review: Review and retry questions you answered incorrectly.

    Modern UI: Clean and minimalistic design with hover effects and rounded buttons.

    Progress Bar: Visualize your progress through the quiz.

This project is ideal for learning Java Swing, event handling, and UI design. It can also be extended with additional features like more questions, a database for storing results, or even a timer for each question.
